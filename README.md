# Website Para Agência de Design

Projeto criado para revisar e praticar habilidades com HTML, CSS (em especial) e Javascript.

## Sobre o Projeto

Será desenvolvido um website institucional de página única com várias seções, como:

* Seção de destaque;
* Seção sobre a empresa;
* Seção de serviços principais;
* Seção de últimos projetos;
* Seção para opiniões de clientes;
* Rodapé com campo para envio de email, botões de redes sociais etc.

Este website será inspirado em um projeto do Vektora Studio, um estúdio criativo global de design focado em UX/UI, branding e desenvolvimento de sites. Você pode conferir o projeto no Dribbble clicando a seguir: [projeto no Dribbble](https://dribbble.com/shots/21984230-Wavessly-Digital-Agency-Website).

## Objetivos de aprendizado

* Estruturação de página HTML5 e sua estilização com CSS3
* Organização de arquivos
* Utilização de fontes externas (Google Fonts)
* Manipulação de imagens
* Versionamento com Git
* Publicação no GitHub
